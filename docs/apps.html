Vowrite for macOS

A menu-bar companion that types for you. Press a hotkey, speak, and clean text appears at your cursor — in any app.

macOS 14+ Apple Silicon Free · Open Source

Install in 3 steps

1

Download the DMG

Grab the latest signed build from GitHub Releases. ~5 MB, EdDSA signed, auto-updates via Sparkle.

Download Vowrite.dmg →
2

Drag to Applications

Mount the DMG, drop the app into /Applications. First launch will prompt for Microphone + Accessibility permission — grant both (the second enables Cmd+V keystroke output).

3

Add an API key

Open Settings → Providers → paste your Groq + DeepSeek (or OpenAI) keys. Stored in macOS Keychain. No account, no telemetry.

Vowrite for iOS

A system-wide keyboard extension that records, polishes, and types — the same engine you use on Mac, in your pocket.

iOS 17+ A14+ Free · Open Source
📲

VowriteIOS app

Hosts the engine, settings, history, Modes, and language pickers. Runs the recording service in the background so the keyboard can stay tiny.

⇄
⌨️

VowriteKeyboard extension

A real iOS keyboard. Tap the mic to dictate, long-press to pick Translate. Talks to the host app over App Group + Darwin notifications.

Install (build from source)

⚠️ The iOS app is open source but not yet on the App Store. For now you'll build it yourself in Xcode — or wait for TestFlight (coming).

1

Clone the repo

Requires Xcode 15+ and a free Apple developer account.

git clone https://github.com/Joevonlong/Vowrite.git
cd Vowrite/VowriteIOS
open VowriteIOS.xcodeproj
2

Sign and run

In Xcode → Signing & Capabilities → set your Team. Plug in your iPhone, hit ▶. Build & deploy both VowriteIOS and VowriteKeyboard targets.

3

Enable the keyboard

iOS Settings → General → Keyboard → Keyboards → Add New Keyboard → Vowrite. Tap "Vowrite" again and toggle Allow Full Access (required for clipboard + network).
